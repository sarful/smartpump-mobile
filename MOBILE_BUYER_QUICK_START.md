# PumpPilot Mobile Buyer Quick Start

1. Set backend URL in `.env`:
   - `EXPO_PUBLIC_API_BASE_URL=...`
   - Use the real hosted backend URL. The app will fail fast if this is missing.
2. Install:
   - `npm install`
3. Validate:
   - `npm run preflight`
4. Run:
   - `npm run start`
5. Login with your backend user/admin/master accounts.

For full setup: `MOBILE_CODECANYON_INSTALL.md`.
